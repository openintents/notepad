# Application:
Bloc de notas OI

# Promo text:
El Bloc de notas OI permite crear, editar y eliminar notas.

# Description:
OI Notepad es una aplicación de código abierto para escribir notas. Usted puede etiquetar, buscar y enviar sus notas, además de otras funciones básicas, como crear, eliminar y editar notas. Características avanzadas como la encriptación, importación/exportación, y las notas de voz están disponibles a través de extensiones. Todas estas funciones se organizan en un diseño simple y eficiente, personalizable a través de una gran variedad de temas. 

Esta aplicación no tiene anuncios y no requiere permisos de internet. 

El código fuente de esta aplicación libre y de código abierto está disponible en: 
http://code.google.com/p/openintents

Para la lista completa de cambios y la lista de preguntas frecuentes, por favor visite: 
http://www.openintents.org

Usted puede ayudar a mejorar la traducción de su idioma en Launchpad: 
https://translations.launchpad.net/openintents/trunk

Características: 
* Añadir, editar y eliminar notas.
* Pulse prolongadamente en las notas para añadir etiquetas.
* Cambiar el tamaño de fuente y el orden.
* Un diseño estructurado de forma clara con una gran variedad de temas.
* Buscar rápidamente en las notas.
* Mandar notas  a través de MMS o SMS.

Extensiones disponibles para "OI Notepad". 
* Grabar y adjuntar mensajes de voz a las notas con "OI Voice Notes".
* Importar/exportar notas de la tarjeta SD con "OI Convert CSV".
* Mostrar la información de licencia en "OI About".
* Insertar la fecha y la hora en las notas con "OI Insert Date".
* Girar texto del revés con "Upside Down"

Aplicaciones que funcionan con "OI Notepad": 
* Hacer copias de seguridad de notas con "My Backup Pro"
* Cifrar las notas con "OI Safe".
* Gestionar notas en la tarjeta SD con "OI File Manager".
* Actualizar con "OI Update".
* Compartir notas a través de "Facebook", "Gmail", "Google+", "Twitter", y otras aplicaciones.
