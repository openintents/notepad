# Application:
OI Poznámkový blok

# Promo text:
Vytvořte a sdílejte poznámky s OI Poznámkový blok

# Description:
OI Poznámkový blok je open source aplikace pro správu poznámek. S OI Poznámkovým blokem můžete tagovat (štítkovat), hledat a odesílat své poznámky, které s ním předtím vytvoříte, smažete nebo upravíte. Pokročilé vlastnosti jako šifrování, import/export a hlasové poznámky jsou všechny dostupné pomocí rozšíření. Toto všechno Vám přinášíme zabalené v jednoduchém a efektivním rozhraní přizpůsobitelné množstvím motivů. 

Tato aplikace je bez reklam a nevyžaduje přístup k internetu. 

Zdrojové kód této open source aplikace je dostupný na stránkách: 
http://code.google.com/p/openintents

Pro úplný seznam změn a seznam často kladených dotazů navštivte prosím: 
http://www.openintents.org

Můžete také pomoci zlepšit překlad do Vašeho jazyka na Launchpadu: 
https://translations.launchpad.net/openintents/trunk

Vlastnosti: 
* Přidávat, upravovat a mazat poznámky.
* Dlouze stiskněte na poznámky pro přidání tagů.
* Změnit velikost písma a pořadí řazení.
* Intuitivní rozhraní s množstvím motivů.
* Rychle prohledávat poznámky.
* Zasílat poznámky jako MMS nebo SMS.

Dostupná rozšíření pro OI Poznámkový blok: 
* Nahrávat a připojovat hlasové poznámky s OI Hlasovými poznámkami.
* S OI Převést na CSV můžete importovat/exportovat poznámky z SD karty.
* Zobrazit licenční infomace s OI Infem.
* Vložit datum a čas do poznámek s OI Vložit datum.
* Otočit text Seshora dolů se "Seshora dolů".

Aplikace, které pracují s OI Poznámkovým blokem: 
* Zálohujte poznámky s "My Backup Pro".
* Šifrujte poznámky pomocí OI Sejfu.
* Spravujte poznámky na SD kartě s OI Správcem souborů.
* Aktualizovat s OI Update.
* Sdílejte poznámky na Facebook, Gmail, Google+, Twitter a mnoho jiných aplikací.
